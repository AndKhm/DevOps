### Установка Ansible
```
# Обновить репозитарии
sudo apt update
sudo apt -y upgrade

# Проверка версии пакета
python3 -V

# Проверка, установлен ли менеджер пакетов pip
python3 -m pip -V

# Установка менеджера пакетов pip (ели не установлен)
sudo apt install -y python3-pip

# Установка Ansible
pip3 install ansible

```
